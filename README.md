# Ejecute in Development mode

1. clonar repositorio

2. Use comand for donwload package
```
npm i
```

3. Install nest CLI
```
npm i -g @nestjs/cli
```

4. Up database
```
docker-compose up -d
```

5. Rebuild database whit seed
```
http://localhost:3000/api/v2/seed
```

## Stack used in this proyect
 * MongoDB - Moongose - MongoAtlas.
 * Nest.
 * Typescript.
